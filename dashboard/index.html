The Experiment

The Antigravity Prompt Showdown is an experiment in AI code generation. Each of the 6 frontier AI models received the same prompt and were tasked with creating a complete, runnable project showcasing GPU computing, visualization, and research automation.

All projects are optimized for NVIDIA RTX 4070 (12GB VRAM) and demonstrate cutting-edge techniques from CUDA kernels to WebGL shaders to quantum circuit simulation.
